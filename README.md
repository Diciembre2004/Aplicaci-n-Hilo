# App-Hilo
Una aplicación para teléfono celular usando Nodejs, React Native, Express, MySql y PHPMyAdmin. 

La aplicacion sirve para recordatorios de medicamentos. Cuando se crea una cuenta, se tiene disponible crear un recordatorio para los 7 dias de la semana,
eligiendo el tipo de dosis y la fecha de cuando empieza y termina el tratamiento. 

Se utiliza la libreria React Native. 

Se uso el tutorial https://www.youtube.com/watch?app=desktop&v=HMKVnwlhJO0
